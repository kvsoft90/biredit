Compatibility

 Program is intended to work under Windows 2000/XP/2003/Vista/2008/7.

Features

 Brief description of general features:
  • Support of Unicode codepages (UTF-16 little endian, UTF-16 big endian, UTF-8, UTF-7);
  • Support of DOS/Windows and Unix newline formats;
  • Multi-level undo;
  • Fast search/replace text strings;
  • Syntax highlighting;
  • Printing of a document;
  • Support of language files.

Command line parameters

 BirEdit.exe [options] "file.ext"
  • /p
    		•  - print
  • /n
    		•  - silently create new file, if doesn't exist
  • /c
    		•  - paste clipboard contents
  • /q
    		•  - stop parsing command line parameters and close program
  • /u
    		•  - remove file associations with BirEdit
  • "file.ext"
    		•  - file path (in quotes)
