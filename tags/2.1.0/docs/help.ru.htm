Замечание о совместимости

 Программа предназначена для работы в системах Windows 2000/XP/2003/Vista/2008/7.

Возможности

 Коротко об основных возможностях:
  • Работа с кодировками Unicode (UTF-16 little endian, UTF-16 big endian, UTF-8, UTF-7);
  • Работа с DOS/Windows и Unix форматами перевода строки;
  • Многоуровневый откат действий;
  • Быстрые поиск/замена строк текста;
  • Подсветка синтаксиса;
  • Печать;
  • Поддержка локализации.

Параметры командной строки

 BirEdit.exe [параметры] "file.ext"
  • /p
    		•  - печать
  • /n
    		•  - создавать новый файл, если он не существует
  • /с
    		•  - вставить содержимое буфера обмена
  • /q
    		•  - закончить обработку параметров командной строки и выйти из программы
  • /u
    		•  - удалить файловые ассоциации с BirEdit
  • "file.ext"
    		•  - путь к файлу (в кавычках)
